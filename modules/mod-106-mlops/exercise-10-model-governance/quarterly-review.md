# Quarterly Model Review — Cadence + Agenda

Quarterly (~90 min). Attendees: ML eng leads, compliance, ds, PM.

## Agenda

1. Inventory: every Production model with current Card + Decision Log status (5 min)
2. Drift dashboard review: which models have PSI > 0.2 (10 min)
3. New bias reviews completed this quarter (10 min)
4. Retraining schedule changes (10 min)
5. Outstanding governance debt: missing cards, decisions undocumented (15 min)
6. Approvals needed for the coming quarter (15 min)
7. Action items (15 min)

## Outputs

- Updated model card per model
- 1 line of decision log per architectural change
- Bias review entry for any new protected feature
- Action list owned by ML lead
